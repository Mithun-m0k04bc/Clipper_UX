CLIPPER - Product Details Page Demo
====================================

How to View:
------------
1. Unzip this folder
2. Open "index.html" in your browser
   - Double-click index.html, OR
   - Right-click → Open With → Chrome/Safari/Firefox

Features Included:
------------------
✅ Living Design System integration
✅ Custom entity icons from Figma
✅ 3-column layout (Left Sidebar, Main Content, Right Sidebar)
✅ Tab navigation (Overview, Usage & Outcomes, Objective Alignment, etc.)
✅ AI Insights panel
✅ Living Design Tags and components
✅ Responsive charts and visualizations

Technical Details:
------------------
- Built with React + TypeScript + Vite
- Uses Tailwind CSS
- Living Design tokens integrated
- Custom Figma icons implemented

Note:
-----
This is a static build. For the full development experience with 
hot-reloading, you'll need to clone the repository and run:
  npm install
  npm run dev
